Speech Command Recognition with CNNs

Ben Bellenchia*, Kai Perez*, Brian Patterson*, Eric Dutan*
UAlbany CNSE ECE465 - Machine Learning Project
Showcase Day 2026
*Indicates Equal Contribution
Progress Report First Update Code Demo

Audio clips are converted into mel spectrogram images, allowing a convolutional neural network to classify spoken commands while rejecting silence and uncertain inputs.

Abstract

This project presents a MATLAB-based speech command recognition system that converts one-second, 16 kHz audio clips into normalized 64 × 64 mel spectrogram images for classification with a convolutional neural network (CNN). The final model, modelv5, classifies the spoken commands “go,” “no,” “stop,” and “yes,” while also detecting silence and using an “unknown” class to reject uncertain inputs. The dataset was split into 70% training, 15% validation, and 15% testing data, with random time shifting, gain variation, and background-noise augmentation added to improve robustness. The final CNN uses three convolution layers with batch normalization, max pooling, dropout, and fully connected classification layers. After the model predicts a command, a 75% confidence threshold is used so low-confidence predictions are labeled as unknown instead of being forced into a command class. This design trades a small amount of clean-command accuracy for a more realistic and reliable speech-command system, reaching about 91.6% validation accuracy while handling noisy, silent, and uncertain inputs.

Model Results

The final CNN model, modelv5, classifies go, no, stop, and yes, while also handling silence and uncertain inputs with an unknown class. Audio clips are one second long at 16 kHz, normalized, converted into 64 × 64 mel spectrograms, and split into 70% training, 15% validation, and 15% testing data.

The final architecture uses three convolution layers with batch normalization, max pooling, dropout, and fully connected classification layers. Earlier four-command models reached about 93.9% accuracy on the cleaner command-only task, while the final model reached about 91.6% validation accuracy after adding noise augmentation, silence detection, and unknown-input rejection for a more realistic speech-command system.

Project Demonstration

The MATLAB demo records a one-second audio sample, normalizes the signal, converts it into a 64 × 64 mel spectrogram, and passes it through the trained CNN to predict the spoken command.

The demo reports the predicted label and confidence score. If the model confidence is below 75%, the result is labeled as unknown rather than forcing the recording into one of the four command classes. This helps the system reject silence, background noise, or speech that does not clearly match a trained command.

Model Progression

Intermediate model: added deeper CNN layers and improved validation accuracy.
